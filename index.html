IT Announcements
Symantec Endpoint Protections Does Not Start

Symantec Endpoint Protection does not start after installing the Windows Patch. The latest Windows patch from Microsoft was a patch to fix this Meltdown/Sceptre malware and it inadvertently caused issues with SEP. When the Windows team realized SEP was not running, they stopped the patch from going out to Win 7 users. For Win 7 users, the security team will push SEP 14 from Monday/Tuesday. Once SEP 14 is installed, those users will receive the Windows patch update that fixes Meltdown/Sceptre malware issue. The Windows patch will go out to Win 10 users from 23-Jan onwards. For Win 7 users, it will be after the user has upgraded to SEP 14. SEP 14 will be distributed via EZUpdate from 22-Jan or 23-Jan.
